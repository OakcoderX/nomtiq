Built for Agent Skills

The repository root contains a portable SKILL.md. Compatible agents discover it from a concise description and load the full workflow only when needed.

Personalized

Recommendations account for taste, budget, neighborhood, party size, noise level, and occasion—not just popularity.

Global

Amap serves mainland-China destinations. Google Maps through Serper serves destinations elsewhere.

Privacy-bounded

Restaurant preferences and visit feedback stay local. The core does not publish reviews, make reservations, place orders, or send messages.

Example: a quiet Beijing date night

Prompt

北京找一家安静、适合约会、人均 300 元以内的餐厅。不要网红店，最好方便聊天。

What the agent does

Clarifies the neighborhood if needed, searches live candidates, filters by budget and conversation-friendly atmosphere, then returns two supported fits plus one clearly labeled exploration choice.

What Nomtiq is for

Nearby restaurant recommendations, date-night restaurants, business dinner venues, family-friendly dining, solo meals, anniversaries, quiet places for conversation, and recording a restaurant visit to improve a local taste profile.

What it is not

Not recipes, grocery planning, calorie tracking, food delivery, restaurant marketing, automated review posting, or a booking agent.
