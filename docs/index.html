Built for Agent Skills

The repository root contains a portable SKILL.md. Compatible agents discover the skill from its name and description, then load the workflow only for restaurant-selection tasks.

Personalized

Recommendations account for taste, budget, neighborhood, party size, noise level, and occasion—not just popularity.

Live and global

Amap serves mainland-China destinations. Google Maps results through Serper serve destinations elsewhere.

Privacy-bounded

Restaurant preferences and visit feedback stay local. The core does not publish reviews, book tables, place orders, or send messages.

Example: a quiet Beijing date night

Prompt

北京找一家安静、适合约会、人均 300 元以内的餐厅。不要网红店，最好方便聊天。

What the agent does

It clarifies the neighborhood if needed, searches live candidates, checks budget and conversation-friendly atmosphere, then returns two supported fits plus one clearly labeled exploration choice.

Restaurant decisions, not generic lists

Use Nomtiq for nearby restaurant recommendations, date nights, business dinners, family-friendly dining, solo meals, anniversaries, quiet places for conversation, and recording a visit to improve a local taste profile.

See activation rules and integration details for AI agents →

Clear boundary

Nomtiq is not for recipes, grocery planning, calorie tracking, food delivery, restaurant marketing, automated review posting, or reservations.
